## List Waitlists

Retrieve all waitlists for your workspace.

<Endpoint method="GET" url="/waitlists" />

### Response

<ResponseField name="waitlists" type="array" required>
  Array of waitlist objects
</ResponseField>

<ResponseField name="count" type="number" required>
  Total number of waitlists
</ResponseField>

### Example Request

```bash
curl https://www.referralloop.dev/api/v1/waitlists \
  -H "Authorization: Bearer pk_live_YOUR_API_KEY"
```

### Example Response

```json
{
  "waitlists": [
    {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "name": "Product Launch",
      "slug": "product-launch",
      "description": "Join our waitlist",
      "status": "active",
      "created_at": "2024-01-15T10:00:00Z"
    }
  ],
  "count": 1
}
```

### Waitlist Object

<ResponseField name="id" type="string" required>
  Unique identifier for the waitlist
</ResponseField>

<ResponseField name="name" type="string" required>
  Name of the waitlist
</ResponseField>

<ResponseField name="slug" type="string" required>
  URL-friendly identifier for the waitlist
</ResponseField>

<ResponseField name="description" type="string">
  Description of the waitlist
</ResponseField>

<ResponseField name="status" type="string" required>
  Status of the waitlist (`active` or `inactive`)
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO 8601 timestamp when the waitlist was created
</ResponseField>
